相册使用说明
2025-06-06

本主题相册页支持blueimpGallery，需在ejs文件内手动引入相关资源。

使用例

配置

favicon: /images/favicon.ico

menu:
    Archives: /archives
    Categories: /categories
    Gallery: /gallery
    About: /about

article:
    # Show word count and estimated reading time.
    readtime: true
    # Code highlight theme, please see https://highlightjs.org/static/demo/
    highlight: atom-one-dark    

logo:
    text: mykonakona

navbar_links:
  RSS:
    icon: fas fa-rss
    url: https://mykonakona.github.io/atom.xml

search:
    type: insight

# Other plugins and their settings.
plugins:
  mathjax: true
  katex: false
  gallery: true
  clipboard: true

相册页

source下新建/gallery/index.md文件，插入以下内容：

---
layout: "blueimpgallery"
comments: false
---
